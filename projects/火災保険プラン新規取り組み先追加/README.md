# 火災保険プラン新規取り組み先追加

## 概要
現状、顧客にデジタル完結手続きを提供できているのは東京海上日動火災保険株式会社の商品のみであるが、他の商品を提案できるようにすることで利便性向上・安心感の向上を図る。

具体的には、[Finatextの「Inspire」システムを活用した日新火災の「ドコモの賃貸火災保険」](https://prtimes.jp/main/html/rd/p/000000472.000012138.html)を当社が提供している不動産投資マイページで表示可能にする。
