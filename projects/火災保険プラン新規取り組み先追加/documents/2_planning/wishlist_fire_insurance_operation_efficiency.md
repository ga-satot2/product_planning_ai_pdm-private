# 火災保険業務効率化

## エグゼクティブサマリー

**■ なぜ今やるのか（背景と危機意識）**
現状の東京海上日動（以下、東海）を主軸としたオペレーションは、件数増加に比例して「非生産的なコミュニケーション工数」が増大する**非スケーラブルな構造**である。Partnersから人員増抑制の要望がある一方、現行仕様では人力介入が不可避であり、このままではFY30に10名以上の体制が必要となる「利益を食いつぶす事業肥大化」が確定している。

**■ やった時のリターン（構造改革の効果）**
FY27後期より、Web完結フローを持つ日新火災（以下、日新）へ新規契約を原則移行する。これにより、事務作業の約85%を保険会社側へ「構造的に移管」し、**件数が増えても人員を増やさない運営体制**を実現する。FY32までの累計収支で**約2.3億円のプラス**（人件費回避＋営利改善）を創出する。

**■ 投資判断**
初期投資560万円。手数料率は約38%減少するが、回避できる人件費（1,000万円×削減人数）がそれを圧倒的に上回る。1件あたりの「限界利益」を最大化し、労働集約型モデルから脱却する。

## ISSUE（現状の課題）

### ターゲットユーザー
* **PARTNERS（火災保険業務担当者）**：
本来不要な「追っかけ事務（決済案内・変更対応）」に忙殺され、本来の付加価値業務に注力できない現場。

### FACT / Root Cause / Damage
* **FACT（事実）**
    * **労働の内訳**：1日の稼働の約7割以上が「始期日変更・クレカ登録案内・各所調整」等の属人的なコミュニケーション。
    * **物理的限界**：フルタイム3名（約6,000時間）が余力ゼロで稼働中。FY30には10人体制、FY32には14人体制が必要になる見込み。
    * **工数の乖離**：純粋な事務作業（1,695h）に対し、ヒアリングした実効工数（6,015h）は**約3.5倍**。この差分が「コミュニケーション債務」として組織に重くのしかかっている。

* **Root Cause（要因）**
    * **システムの制約**：東海の仕様上、引渡後の始期日変更に伴う「再案内・再計上」が不可避であり、代理店による人力の証拠受領（メール返信等）が義務付けられている。
    * **決済フローの分断**：決済日変更のたびに、顧客・営業・保険会社の間で弊社が「人力のハブ」となる必要がある設計。

* **Damage（影響）**
    * **営利の毀損**：手数料収入の伸びを人件費の増分が追い越し、事業としての経済合理性が喪失する。
    * **組織リスク**：担当者の離脱が即、融資実行（火災保険加入必須）の遅延を招く脆弱な体制。

## USER SCENARIO

### 日新火災への切り替えによる「構造的工数移管」
日新のWeb完結フローを採用することで、業務の主導権と実務負担を「自社」から「保険会社と顧客」の間へ移転させる。

| 項目 | 現行（東京海上） | 改革案（日新火災） | 変化のポイント |
| :--- | :--- | :--- | :--- |
| **事務の主体** | **弊社（代理店）** | **顧客 ↔ 保険会社** | 弊社の介入を排除 |
| **変更対応** | 手動で再計上・再案内 | Web上で顧客が自己完結 | 調整工数の消滅 |
| **督促業務** | 弊社から電話・メール | 保険会社の自動リマインド | コミュニケーション債務の解消 |

## IMPACT（定量的・定性的効果）

### 上位目標 (GOAL)
**「FY30においても現状の3人体制以下での運営」を実現**

* **工数削減**：年間約4,981時間（実効工数の約88%を削減）
* **リソース解放**：最大13名分のリソースを他事業へ再配置可能
* **BCPの強化**：属人性の排除により、担当者の離脱リスクを最小化

| 指標 | 現状維持 (FY30) | 日新火災移行 (FY30) | 差分 |
| :--- | :--- | :--- | :--- |
| **運営体制** | 10名（コスト1億円） | **3名※（コスト3,000万円）** | ▲7名 / ▲7,000万円 |
| **年間工数** | 5,663時間 | 682時間 | ▲4,981時間 |
| **顧客体験** | 引渡後にアナログ後追い | Webで事前完結 | ストレスゼロへ |

※理論上の事務工数は1名分だが、例外対応および組織の継続性（BCP）を考慮し、3名体制をミニマムとする。

### 工数算出ロジック（FY25実績）

「事務ログ」だけでなく「フルタイム稼働」の実態をベースに算出。

| 項目 | 年間工数 | 備考 |
| :--- | :--- | :--- |
| 事務ログ作業 | 1,695h | 加入申込・計上・保全など |
| コミュニケーション工数（コミュニケーション債務） | 4,320h | メール、電話、社内調整（120h/月 × 3名） |
| **合計実効工数** | **6,015h** | **フルタイム3名分（2,005h/人）に相当** |

**結論**：現在の3名体制はすでに「遊び」がない状態でフル稼働しており、件数増への耐性がゼロ。

- 新規業務：年間契約件数 × 9.4分/件
- 保全業務：累計契約数 × 1.1分/件
- 更新業務：(5年前の申込件数× 0.6) × 10.0分/件
