## 管理プラン（マスターリース契約）

### ライブズナビ
- **管理手数料**: 月額3,000円（修繕保証委託報酬として、賃料と相殺）
- **契約期間**: 5年間

### 明和住販流通センター
- **管理手数料**: 転借人との賃貸借契約の賃料の10%（実質的な管理手数料率）
- **契約期間**: 2年（自動更新）
