# LMS Webアプリ 未実装機能一覧

Web アプリ（`webapp.gs` + HTML）で意図的に未実装またはスタブのみの機能をまとめる。全体の未実装は `lms_restart_analysis.md` の「未実装機能」を参照。

---

## Web アプリ側の実装状況（2026-02-19 更新）

| 機能 | 現状 | 備考 |
|------|------|------|
| **Web からの予約作成** | 実装済み | 参加情報のコース列を「予約済み」に更新。カレンダーにゲスト追加。グループ・コース番号で整合 |
| **Web からのキャンセル** | 実装済み | `cancelReservation(email, eventId)` を呼び出し。マイページから eventId を渡してキャンセル |
| **?page=register（受講者登録）** | 実装済み | `Register.html` と `handleRegister` / `webapp_registerParticipant` で参加情報に1行追加 |
| **管理者の詳細機能** | 実装済み | コース管理（admin-courses）、セッション管理（admin-sessions）、一斉メール案内（admin-email）、レポート（admin-reports）を各ページで提供 |
| **プロフィール更新** | 実装済み | `handleUpdateProfile` で参加情報シートの氏名列を更新。マイページに「氏名を変更」フォームあり |

---

## 既存 GAS／フォーム側で対応しているが Web UI では未実装のもの

- テスト・成績（フォーム／GAS で対応、Web では未実装）
- 予約変更（1回まで・3日前まで）: 既存 GAS にも未実装（`lms_restart_analysis.md` 参照）
- リマインド: Slack のみ実装、メールは未実装

## 全体の未実装（詳細は lms_restart_analysis.md）

- **高優先**: 予約変更（3日前・1回制限・変更履歴・カレンダー更新）、テスト・採点、出席管理、メール通知、管理ダッシュボード（統計・レポート）
